BF 1942 Install Instructions

Windows

  1. Download the game
  2. Run bf1942-hd-setup.exe wait till installation is complete (it also installs anti-cheat program punkbuster) don't start the game.
  3. Run the gamespy patch so you can play online battlefield_1942_gamespy_patch_v1.61.exe
  4. Launch the game
  5. Click multiplay, internet, join server, and enter rct.nolanw.ca and port 14567. Password is peepeedoodoo

Mac

  1. Download the game
  2. Download Play on Mac https://www.playonmac.com/en
  3. Install Play on Mac
  4. Open Play on Mac (right click, open)
  5. Click Install a program
  6. Tick "no cd required", search 1942 and click Battlefield 1942
  7. Click through next with default options
  8. It will ask you to point it to the game files. Unzip BF1942 and select the path for the setup exe
  9. Continue mashing through default options and next. There's one thing that fails but if you do it again it succeeds so do that.
  10. Launch the game from Play on Mac
  11. Click multiplay, internet, join server, and enter rct.nolanw.ca and port 14567. Password is peepeedoodoo

Tips

  • You must be in-game before your resolution changes do anything
  • The menu system is not great, you may need to click around a bit before it does what you want
